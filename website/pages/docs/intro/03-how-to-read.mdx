# 03. フローチャートの見方

<p className="goal">フローチャートの見方を理解する</p>

## 学習

### フローチャートの見方

フローチャートは「開始」の端子から始まり、「終了」の端子で終わります。
開始と終了の間には実行したい処理を実行したい順番どおりに記述し、線でつなげます。
また、処理の流れは原則上から下に向かって流れるように記述します。

## 例題1
「Hello Word!」と出力するフローチャート。

![フローチャートの見方1](./how-to-read1.svg)

### 実行順
> 1. プログラムの開始
> 2. "Hello World!" を出力
> 3. プログラムの終了

### 実行結果

```
Hello World!
```

## 例題2
「Hello」と「Word」を順番に出力するフローチャート。

![フローチャートの見方2](./how-to-read2.svg)

### 実行順
> 1. プログラムの開始
> 2. "Hello" を出力
> 3. "World" を出力
> 4. プログラムの終了

### 実行結果

```
Hello
World
```
