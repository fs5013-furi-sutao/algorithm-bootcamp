# 02. フローチャートで使用する代表的な記号

<p className="goal">フローチャートを記述する際に仕様する代表的な記号を知る</p>

## 記号の一覧

フローチャートを記述する際に使える記号は決まっています。
実行したい処理の種類によってこれらの記号を使い分けます。


以下はその代表的な記号です。

| 記号名 | 記号  | 意味 |
| :-- | :--: | :-- |
| 端子 | ![端子の記法](./start-end.svg) | プログラムの開始・終了を表す記号。フローチャートの開始と終了に使う。 |
| 処理 | ![処理の記号](./process.svg) | 変数の宣言や値の代入などの処理表す記号。 |
| 入出力 | ![処理の記号](./input-output.svg) | データの入出力を表す記号。 |
| 判断 | ![判断の記号](./decision-making.svg) | 条件によって処理を分岐する記号。 |
| ループ | ![ループの記号](./loop.svg) | 処理を繰り返しを表す記号。二つの記号の間に挟んだ処理を繰り返す。 |
| 結合子 | ![結合子の記号](./connector.svg) | 同じフローチャートの他の部分への出口または入口を表す。 |

上記以外にもフローチャートで使われる記号がいくつか存在しています。
本カリキュラムではそれらを使わないため、紹介していませんが、気になる方はぜひ検索をしてみてください。
