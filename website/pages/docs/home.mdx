# アルゴリズムとは

Java Bootcamp は、およそ 250 時間で Java 言語とプログラミングの 基礎 ～ 発展的内容までの知識を手に入れる、フリースタイル ITS 所属の方向けのプログラムです。

## アルゴリズムとは

アルゴリズムを少し硬い言葉で説明をすると、次のようになります。

アルゴリズムとは、問題を解決するための手順や手法のことです。アルゴリズムは、特定の入力を受け取って、その入力に対して決まった手順に従って処理を行い、最終的に望ましい結果を得るための手法です。

## 身の回りのアルゴリズム

現代は「アルゴリズム」という言葉を耳にする機会が多いです。
IT業界以外の人であっても「Goolgeの検索アルゴリズム」や「Youtubeのアルゴリズム」という言葉を聞いたことがあるはずです。

例えば、Googleの検索アルゴリズムの場合、入力した検索ワードに対してどのページをどのような順番で表示すべきかを決めています。
Youtubeも同じように検索結果の内容や順番、おすすめの動画などを決めています。

このように聞くとアルゴリズムとはとても難しいものに聞こえますが、その本質は決して難しいものではありません。

## アルゴリズムの本質

アルゴリズムは単純に特定の目的を達成するための手順を指します。

例えば、料理のレシピを見ながら調理したことがある人であれば、既にアルゴリズムのお世話になっています。
料理のレシピそのものが一つの「アルゴリズム」です。
必要な食材を準備してレシピの手順を厳密に守りながら調理をすれば目的の料理が完成します。

プログラミングでも同じように、特定の目的を達成するためには決まった手順で処理を行います。

本カリキュラムでは、この処理の手順を考える力を身に付けていきます。
