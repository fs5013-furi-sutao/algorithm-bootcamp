# 01. 例題(基本)

<p className="goal">制御構造を組み合わせて複雑な動きを作る</p>

## 例題1

制御構造を組み合わせたフローチャートを確認していく前に配列について学習しましょう。
配列は変数と同じようにデータを保持することができます。
ただし、変数が値を一つ格納することができるのに対して、配列は複数の値を保持することができます。

変数が「箱」に例えられるのに対して、配列は銭湯の「ロッカー」に例えられます。
銭湯のロッカーには番号が割り振られています。
荷物の出し入れをする場合はその番号を確認してから使用します。

配列も同じように値を入れることのできる場所が複数存在しており、
その一つ一つを「要素」と呼ぶ。また、その要素に割り当てられた番号を「要素番号」と呼びます。
要素番号は「0」から始まることに注意をしましょう。

以下は、配列を使ったフローチャートです。「```配列の初期化処理```」は配列に値を入れる処理です。
この処理でフォローチャートの右側にある配列を作成しています。
