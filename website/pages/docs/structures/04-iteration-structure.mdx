# 04. 反復構造とは

<p className="goal">反復構造とは何かを知る</p>

## 反復構造

反復構造とは、指定した条件を満たす間、処理が繰り返す制御構造です。
反復構造には2つの書き方があります。「ループ」記号を使った書き方と「判定」記号を使った書き方です。
本カリキュラムではループ記号を使った書き方を用います。

これらのフローチャートの場合、条件の判定が「真」の間は「処理A」を繰り返し実行します。
