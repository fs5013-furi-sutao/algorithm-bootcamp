# 02. 分岐構造とは

<p className="goal">分岐構造とは何かを知る</p>

## 分岐構造

分岐構造とは、指定した条件によって、処理が分岐する制御構造です。
指定した条件に当てはまる場合は「Yes」、当てはまらない場合は「No」の経路を通る。

プログラミングでは指定した条件が成り立つことを「真」または「true」と表現し、
成り立たないことを「偽」または「false」と表現する。

![分岐構造](./selection-structure.svg)

このフローチャートの場合、条件の判定が「真」の場合は「処理A」を、
条件の判定が「偽」の場合は「処理B」が実行される。
