# 01. 制御構造とは

<p className="goal">アルゴリズムの記述に使う３つの制御構造を知る</p>

## アルゴリズムの3つの制御構造

身の回りにはプログラムによって制御されているものが数多く存在する。
パソコンやスマートフォン、スマートウォッチ、エアコンなどはすべてプログラムに制御されている。
これらの機器を制御するプログラムが複雑なものであることは容易に想像ができるはずです。

しかし、どんな複雑なプログラムでもたった3つの制御構造の組み合わせでできている。
3つの制御構造とは「順次構造」「分岐構造」「反復構造」です。

| 順次構造 | 分岐構造  | 反復構造 |
| :--: | :--: | :--: |
| ![順次構造](./sequence-structure.svg) | ![選択構造](./selection-structure.svg) | ![反復構造](./iteration-structure-type1.svg) |

プログラムが書けるようになるためにはこれらの制御構造を理解し、効率よく組み合わせる必要がある。
