<p className="goal">分岐構造の使い方を知ろう</p>


## 例題6

次のフローチャートでは分岐構造の中にもう一つの分岐構造が使われています。
このような構造のことを「入れ子構造」または「ネスト構造」と呼びます。

![例題6](./selection-example-006.svg)

<Question question = "変数 number = 17 の場合の実行結果" opened="true">
    <Answer
        imgSrc = "./selection-example-006_3.svg"
        answer = "どちらも割り切れない"
    />
</Question>

分岐構造を複数組み合わせることでより複雑な判定を行うことができる。。
例えば、このフローチャートでは入力した値が「```2で割り切れる```」「```3で割り切れる```」または「```どちらでも割り切れない```」かを出力します。

<Question question = "変数 number = 6 の場合の実行結果" opened="true">
    <Answer
        imgSrc = "./selection-example-006_1.svg"
        answer = "2で割り切れる"
    />
</Question>

<Question question = "変数 number = 27 の場合の実行結果" opened="true">
    <Answer
        imgSrc = "./selection-example-006_2.svg"
        answer = "3で割り切れる"
    />
</Question>

## 例題7

次のフローチャートは例題6と比較して内容が少し似ています。
入力した数値が2で割り切れるか、3で割り切れるかをチェックしているところは変わりません。
しかし、ネスト構造が変わったことで、2と3の両方で割り切れることがチェックできるようになりました。
